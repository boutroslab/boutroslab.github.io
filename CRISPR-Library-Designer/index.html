    CRISPR Library Designer

    less than 1 minute read

    The CRISPR library designer (CLD) is an integrated software application for the design of single guide RNA (sgRNA) libraries for all organisms with annotated genomes. CLD is suitable for the design of libraries using modified CRISPR enzymes and targeting non-coding regions. The software is an end-to-end workflow for the fully automated design of small and large-scale CRISPR libraries.

    CLD predicts a high fraction of functional sgRNAs and is publicly available at https://github.com/boutroslab/cld
